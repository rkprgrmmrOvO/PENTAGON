Greetings! My name is Mark Angelo Malano, and I am from BMMA401. Learning about JavaScript has been a truly enjoyable experience. JavaScript is a programming language designed to make web pages interactive. It can perform a wide range of tasks, such as showing alerts, creating animations, and updating content dynamically without needing to reload the entire page. In essence, JavaScript is what transforms static HTML and CSS into engaging, responsive user experiences, making websites dynamic and fun to use.



What I liked most about learning JavaScript was its ability to manipulate and move the content of a website seamlessly.
